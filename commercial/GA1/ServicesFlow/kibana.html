Kibana

Kibana is an open source Apache licensed, browser-based analytics and search dashboard for ElasticSearch.
Using Kibana, you can add visualizations to your data obtained through ElasticSearch. Create dashboards, graphs, charts, and other visualizations. Take your log files and sort, filter, query, and more. Then create your charts to help you create usable information from mounds of incomprehensible text data.
Using Kibana in HP Helion OpenStack cloud implementations will help you forecast, plan, and use the data you collect.
