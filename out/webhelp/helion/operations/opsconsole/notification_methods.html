This page allows you to define notification methods that are used by the alarms. By default, notification methods are sorted by name and in a table format.

Search and Sort

The search bar allows you to filter the notification methods you want to view.

The checkbox allows you to select (or deselect) a group of notification methods to delete:

  • Select Visible allows you to delete the selected notification methods from the table.
  • Select All allows you to delete all the notification methods from the table.
  • Clear Selection allows you to clear all the selections currently selected from the table.

You can display the notification methods by grid, list or table views by selecting the corresponding icons next to the Sort By control.

The Sort By control contains a drop-down list of ways by which you can sort the alarm definitions. Alternatively, you can also sort using the column headers in the table.

  • Sort by Status displays the notification methods by their status.
  • Sort by Metric displays the descriptive details of the alarm definitions.
  • Sort by Severity displays the alarm definitions' notification criteria, such as whether enabled for certain types of alarms and how the notification is disseminated (by Email, Web, etc.)

Create Notification Methods

The Create Notification Methods button beside the search bar allows you to create a new notification method.

To create a new notification method:

  1. Click the Create Notification Method button to open a separate window that allows you to configure a new notification method.
  2. In the Create Notification Method window, type a name for the notification in the Name text field. The name is required and may be up to 255 characters in length consisting of letters, numbers, or special characters.
  3. Select a Type in the drop down. Choices are:.
    • URL allows you to enter in an internet address Also referred to as a Web Hook.
    • Email allows you to enter in an email address. See Note Below.
    • PagerDuty allows you to enter in a PagerDuty address.
  4. In the Address/Key text field, enter in a valid value based on the type you chose.
  5. When done creating the notification method, click Create Notification Method.
Note: For this method to work you need to have a SMTP server specified.

After the notification method saves, the Notification Methods window refreshes with the new notification method populated in the table.
