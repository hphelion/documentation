Create Chart

The Create Chart button beside the search bar allows you to create a new data chart.

To create a new chart:

  1. Click the Create Chart button to open a separate window that allows you to configure a new chart.
  2. In the Chart Definition section you will make the following choices:
    • Give your chart a Name.
    • Select a Chart Type in the drop down. Choices are:
      • Line
      • Bar
      • Stacked Bar
      • Area
      • Stacked Area
    • Select a Chart Size in the drop down. Choices are:
      • Small
      • Large
    • Select a Chart Update Rate in the drop down. Choices are:
      • 1 Minute
      • 5 Minutes
      • 10 Minutes
      • 30 Minutes
      • 1 Hour
    • Select a Time Range in the drop down. Choices are:
      • Last 60 Minutes
      • Last 2 Hours
      • Last 4 Hours
      • Last 8 Hours
      • Last 24 Hours
      • Last 7 Days
      • Last 30 Days
      • Last 90 Days
  3. In the Chart Data section you will select up to six metric and dimension pairs to define what information your chart will reflect.
    • Select a Metric from the drop down.
    • Select an Associated Dimension from the following options:
      • Service
      • Region
      • Avail. Zone
      • Host
      • Device
    • Select the Add Data to Chart button to add the metric and dimension pair to the list that populates on the right side
  4. When done creating the chart, click Create Chart to populate it in the Metric Monitoring chart list.

After the chart saves, the Metric Monitoring window refreshes with the new chart data populated in the view.
