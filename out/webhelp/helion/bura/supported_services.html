By default, backups include the following:
  • All important information on the deployer
  • MySQL database
  • Swift rings
The following services will be effectively backed up. The critical data that will be backed up are the databases and the configuration related files. Service-specific limitations are also listed.
  • Ceilometer - No backup of metrics data
  • Cinder - No backup of the volumes
  • Glance - No backup of the images
  • Heat
  • Horizon
  • Keystone
  • Neutron
  • Nova - No backup of the images
  • Swift - No backup of the objects
  • Operations Console
  • Monasca - No backup of the metrics

Limitations

  • Recovery of the following services (or cloud topologies) will be partially supported as they need additional data (other than MySQL DB) to return to fully functional.
    • ESX Cloud
    • Ceph Cloud
    • Network services - LBaaS and VPNaaS
  • Logging data (that is, log files)
