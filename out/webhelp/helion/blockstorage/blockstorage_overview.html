About Block Storage

HPE Helion OpenStack Block Storage Volume Operations uses the OpenStack Cinder (block storage) service to manage storage volumes, including creating volumes, creating volume snapshots, and configuring volumes.

Once created, you can disconnect block storage volumes from one instance and attach them to another. And you can create snapshots of these volumes.

Helion OpenStack 2.0 supports the following storage back ends for block storage volumes and backup datastore configuration:

  • Volumes
    • VSA; for more information, see HPE Helion OpenStack 2.0: Configuring for VSA Block Storage Backend.
    • 3PAR FC or iSCSI; for more information, see HPE Helion OpenStack 2.0: Configuring for 3PAR Block Storage Backend.
    • VMDK; for more information, see Modify the Volume Configuration File.
    • Ceph; for more information, see HPE Helion OpenStack 2.0: Ceph Overview.
  • Backup
    • Swift
    • Ceph
  • Zone Driver
    • Brocade Zone Manager; for more information, see HPE Helion OpenStack 2.0: Configuring Brocade Fibre Channel Zone Manager.
